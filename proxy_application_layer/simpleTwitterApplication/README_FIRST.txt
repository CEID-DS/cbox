To execute the application you need:
* Install Android SDK 
* Create Android Virtual Machine
* Add to build PATH the .jars contained in twitterAPI folder
* Add manually AccessToken and AccessTokenSecret from twitter4j.properties
  to Android Emulator Storage. ConsumerKey and ConsumerSecret are stored
  in twitter4j.properties of Android Emulator Storage.
  HIT: Use method storeAccessToken() in Authorization.class
